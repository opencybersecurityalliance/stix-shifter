## Azure Log Analytics
### Supported STIX Operators
*Comparison AND/OR operators are inside the observation while observation AND/OR operators are between observations (square brackets).*

| STIX Operator | Data Source Operator |
|--|--|
| AND (Comparision) | and |
| OR (Comparision) | or |
| = | == |
| != | != |
| IN | in~ |
| MATCHES | matches |
| > | > |
| >= | >= |
| < | < |
| <= | <= |
| LIKE | contains |
| OR (Observation) | OR |
| AND (Observation) | OR |
| <br> | |
